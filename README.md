## 💻 How It Works

- The stopwatch uses **JavaScript timing functions** like `setInterval()` to track time.
- Time is updated every second and displayed dynamically.
- Users can:
  - Start the timer
  - Pause it anytime
  - Reset it to zero
  - Save lap times
